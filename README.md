## Описание предметной области

Реализация древнегреческого шифра намотки ленты на цилиндр.

### Основные действия
- Параметр – диаметр цилиндра;
- Опциональная визуализация процесса.
### Математическая основа
- `text.length` - длина текста
- `diam` - количество строк
- `cols` - количество столбцов
- `int index = j * cols + i` - индекс каждого символа в составленной матрице
