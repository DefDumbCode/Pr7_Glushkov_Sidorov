# Pr7_Glushkov_Sidorov
## Буквы
Код представляет собой консольное приложение выводящее построчно "Hello, <строка>! Count to <счетчик>" с добавлением символов к строке и увеличением счётчика.
Использованные методы отладки:
- Использование точек останова и пошаговое выполнение
- Инспекция данных с помощью информационных подсказок
- Проверка переменных с помощью окон "Авто" и "Локальные"
