# Pr7_Glushkov_Sidorov
## Fibonacci
Код представляет собой калькулятор чисел Фибоначчи.
На вход даётся индекс в списке последовательности.
Результатом выполнения функции является значение в последовательности Фибоначчи.

В исходном коде была ошибка, из-за которой функция вычисляла значение предшествующее необходимому.
Использованные методы отладки:
- Использование точек останова и пошаговое выполнение]
